# Design Standards

## Styling
- Use Tailwind CSS with custom animations
- Follow a glassmorphism / neomorphism aesthetic
- Use Framer Motion for all animations (fade, slide, stagger)
- Prefer gradients over flat colors

## Animations
- Page transitions: 0.4s ease-in-out
- Hover effects on all interactive elements
- Scroll-triggered animations using Intersection Observer
- Micro-interactions on buttons and cards

## Libraries to use
- Framer Motion (animations)
- GSAP (complex sequences)
- AOS (simple scroll effects)
- Lottie (JSON-based micro-animations)
- Tailwind CSS (styling)
- Lucide React (icons)
